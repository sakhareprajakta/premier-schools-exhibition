# Premier Schools Exhibition Website

A responsive and modern landing page developed as an assignment for the **Premier Schools Exhibition**. The website is designed to provide information about the exhibition, participating schools, event highlights, and contact details with an attractive and user-friendly interface.

## 🌐 Live Demo

https://sakhareprajakta.github.io/premier-schools-exhibition/

## 📋 Project Overview

This project demonstrates responsive web design principles and front-end development skills by creating an informative website for a school exhibition event. The website focuses on clean UI, easy navigation, and an engaging user experience.

## ✨ Features

- Responsive Design (Desktop, Tablet & Mobile)
- Modern and Clean User Interface
- Hero Section
- About Exhibition Section
- Featured Schools Section
- Event Highlights
- Gallery Section
- Contact Section
- Smooth Navigation
- Interactive Buttons
- Optimized Images and Layout

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/sakhareprajakta/premier-schools-exhibition.git
```

### Navigate to Project Folder

```bash
cd premier-schools-exhibition
```

### Run the Project

Simply open `index.html` in your browser.

Or, if using VS Code:

- Install the **Live Server** extension.
- Right-click on `index.html`.
- Select **Open with Live Server**.
